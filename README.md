## Start Project {React + NodeJS}

First Step <br>

Open new terminal then write <br> 
1-) <code> **cd notes-app-backend** </code> <br>
2-) <code> **node server.js** </code> <br>

Second Step <br>

Open new terminal then write <br> 
1-) <code> **cd notes-app-frontend** </code> <br>
2-) <code> **npm start** </code> <br>
